# Running on Deno

It's possible to use SQLite with Deno by using [`kysely-deno-sqlite`](https://gitlab.com/soapbox-pub/kysely-deno-sqlite).
This method uses [deno-sqlite](https://github.com/dyedgreen/deno-sqlite) under the hood.

```ts
import { Kysely } from 'npm:kysely';
import { DB as Sqlite } from 'https://deno.land/x/sqlite/mod.ts';
import { DenoSqliteDialect } from 'https://gitlab.com/soapbox-pub/kysely-deno-sqlite/-/raw/main/mod.ts';

const db = new Kysely({
  dialect: new DenoSqliteDialect({
    database: new Sqlite('db.sqlite3'),
  }),
});
```
